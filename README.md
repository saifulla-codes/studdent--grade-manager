## About The Project

A Python-powered student grade management system that calculates grades, generates report cards, and provides academic insights! Perfect for GSOC Python learning journey!

## Features

- Subject-wise mark entry
- Automatic grade calculation 
- Beautiful report card generation
- Achievement badges
- Input validation
- User-friendly interface
